Cloudy with a Chance of Ice Cream

Computer Game Written in Python

May 2017 Allison

Cloudy with a Chance of Ice Cream is a game written in Python in which the player moves an ice cream cone back and forth along the x-axis using the arrow keys in an attempt to catch ice cream scoops falling vertically. Obstacles, such as asteroids, will also fall vertically, and the player will need to avoid catching the obstacles lest they sustain damage. It will have one environment, outer space, which will be represented by an appropriately themed background image.

Magic Mirror

For this project, I built a raspberry pi powered smart mirror that displays the time, weather, among other information.

May 15, 2017 Allison

My passionate pursuit project featured building a ‘smart’ mirror. Essentially, a ‘smart’ mirror is a monitor that is programed to display certain information. For example, time, the weather, and news. The mirror element, allows me to use the monitor as a display and as home furniture. This project incorporated several elements that all matched my learning goals. My learning goals were to learn how to design and build a wood frame, learn JavaScript enough to understand the program, and write a document to guide others in the same project. Altogether, I met my learning goals and produced a working smart mirror and accompany guide.

Cantenary Curve Reserach

February 15, 2017

This study was concerned with the investigation for the measurements and curve of the “tread” that would allow a rectangle to roll smoothly about an axis. Previous studies have found the necessary equations for a square, among other shapes, but not a rectangle. The researcher used a combination of arc length, trigonometry, and catenary curves to find if a rectangle could roll smoothly on alternating inverted curves or “tread.”
